# Fertilizer Prediction using Simple Linear Regression

## Objective
The objective of this project is to predict potato yield using fertilizer data with Simple Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset Information
The dataset contains fertilizer values as the independent feature and potato yield as the dependent variable.

## Project Steps
- Import Required Libraries
- Load Dataset
- Separate Feature and Target Variable
- Split Dataset into Training and Testing Sets
- Train Simple Linear Regression Model
- Predict Results
- Visualize Regression Line
- Compare Actual and Predicted Values

## Model Used
- Simple Linear Regression

## How to Run the Project
1. Install required libraries
2. Open the Jupyter Notebook
3. Run all notebook cells sequentially

## Output
The model predicts potato yield based on fertilizer values and visualizes the regression line with actual data points.
